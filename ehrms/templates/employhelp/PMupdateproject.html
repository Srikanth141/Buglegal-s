How can the Project Manager update or modify the details of an existing project or task in the HRMS portal?

To update or modify the details of an existing project or task in the HRMS portal, follow these steps:


You will see the following:

  1. Log in to your HRMS portal using your username and password.
  2. Once log in, you will see the main dashboard.
  3. In That Dashboard you see project team details so, you have a view and edit buttons to modify your existing project.
  4. You can manage your active projects by edit option in the dashboard.
