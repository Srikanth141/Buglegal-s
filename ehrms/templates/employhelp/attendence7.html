What are my options for maintaining leave and attendance?

We provides three options -

1. Use our web base attendance - This is included in the default Hrms subscription, and can be enabled or disabled from Settings.

2. Use our API - If you have your internal mechanism for maintaining leave and attendance, but would like to synchronize that data with Hrms, then you can use our API to do it in an automated manner. For more details on this, please check Settings > API.
