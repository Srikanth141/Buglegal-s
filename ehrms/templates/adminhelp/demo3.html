HRMS - User Roles and permissions

Modify permissions of default user roles

By default, HRMS has 6 user roles.

Their are

1. Administrator
2. Human Resources
3. HRMS
4. Report Viewer
5. Reimbursements
6. Chartered Accountant

User roles can be found under Settings > User roles.

You can either modify permissions in the default roles or create a custom role with custom permissions.

To create a custom role and assign specific permissions.

To modify permissions inside a default role follow the steps below

Step 1: Under Settings | User roles | Click the standard role

Step 2: Modify the permissions for the standard role

Select/deselect the permissions you want to assign to this role and click save changes.

Assigning the role to employees once it is successfully edited

1. Please ensure that the person is added to your organization. This can be either as an employee or a contractor.

2. Go to that person's profile from the People page. Scroll to the User Role / Privileges section, and click on Assign New Privileges.

3. From the drop-down, select the user role that you would like to assign to that person.
