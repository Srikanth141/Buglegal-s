How does Admin create a project?

You will see the following:

  1. Login into Buglegal HRMS portal using your username and password.
  2. Once log in, you will see the main dashboard.
  3. In That Dashboard you see admin options.
  4. Go to 'Admin Options' and there you will see 'Create Project'.
  5. Click on 'Create Project',fill the required fields and clicking on submit will create a project the create project.
