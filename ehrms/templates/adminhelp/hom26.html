How can Admin check employee's performance?

You will see the following:

    How can Admin check employee's performance?

  1. Once log in, you will see the main dashboard.
  2. Click on Employee Performance > Click on the Daily Employee Task Report and it shows the employee's daily task performance report.

  1. Click on the employee average performance report which shows the average performance of the entire project.

  1. Click on the employee project performance select the particular project name then it shows the performance of every employee.

  1. Click on employees task performance, it shows the Daily Employee Project Performance Report data.
